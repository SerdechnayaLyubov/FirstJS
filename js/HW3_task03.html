Задание 3. Создайте функцию, которая принимает на вход массив или объект, подобный массиву.
Функция должна возвращать массив, полученный из аргумента удалением триплов.
Триплом будем считать элемент, которые повторяется 3 и более раз.
Пример:
let a = f([1, 2, 1, 5, 1, 5, 1, 4]); // a = [2, 5, 4]
let b = f("mama ramu"); // b = [" ", "r", "u"];
